### Unity Terrain Generator Plugin

Developed a powerful Unity plugin that generates real-world terrains using online geospatial data sources. The tool fetches imagery and elevation data from servers to automatically create accurate Unity terrain assets.

**Key Features:**
- Fetch satellite/aerial imagery from online tile servers
- Download elevation/heightmap data from terrain services
- Automatic Unity terrain generation with proper scaling
- Support for multiple coordinate systems and projections
- Configurable LOD and resolution settings
- Editor tools for easy terrain customization
